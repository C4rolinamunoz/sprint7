Este proyecto carga un archivo con datos de anuncios de venta de coches y muestra un análisis básico usando Streamlit.

Al ejecutar la aplicación, se visualiza el contenido completo del archivo vehicles_us.csv. Hay un botón que dice "Analizar el DataSet". Al presionarlo, se generan tres gráficos:

Un histograma de precios

Un gráfico de dispersión que relaciona condición y precio, con colores según el modelo del coche

Otro gráfico de dispersión entre kilometraje (odometer) y precio
